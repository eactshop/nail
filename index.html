1. Tạo một Google Sheet mới → mở Extensions → Apps Script
2. Dán đoạn code Apps Script ở cuối trang này vào → Deploy → New deployment → Web app (Execute as: Me, Access: Anyone)
3. Copy URL deployment → thay vào biến GOOGLE_SCRIPT_URL trong script bên dưới
4. Thay NOTIFY_EMAIL bằng email muốn nhận thông báo
